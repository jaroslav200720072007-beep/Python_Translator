# Консольний Англо-Український та Україно-Англійський Словник

Простий та ефективний консольний додаток на Python для двостороннього перекладу слів (з англійської на українську та навпаки) із можливістю динамічного додавання нових слів у базу даних.

## 🚀 Функціонал

* **Двосторонній пошук:** Програма автоматично розпізнає, якою мовою (англійською чи українською) введено слово, і шукає переклад.
* **Регістронезалежність:** Пошук працює однаково добре, незалежно від того, чи введено слово з великої, чи з малої літери (`Apple`, `apple`, `APPLE`).
* **Динамічне оновлення:** Якщо слова немає в базі, програма запропонує додати його. Нове слово одразу стає доступним для пошуку в поточному сеансі та зберігається у файл.
* **Захист від помилок:** * Програма не ламається, якщо текстового файлу бази даних ще не існує (вона створить його автоматично).
    * Ігноруються випадкові порожні рядки під час введення.
    * Коректно обробляються рядки у файлі, які містять кілька знаків `=` (завдяки `split('=', 1)`).

---

## 📂 Структура проєкту

* `main.py` (або ім'я вашого скрипта) — основний код програми.
* `dictionary.txt` — текстовий файл, який виконує роль бази даних. Створюється автоматично при першому додаванні слова.
